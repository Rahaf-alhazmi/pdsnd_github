### Description
Bike share system allow users to rent bicycles for short time, so in this 
project I analyzed the dataset for the first six months of 2017 
on three major cities:Chicago, New York City,and Washington. 
The dataset in all three csv files contain the same core six (6) columns which are:
Start time, End time ,Trip duration, start station, end station and User type.
Also Gender and Birth year columns just on Chicago and New York City datasets.
